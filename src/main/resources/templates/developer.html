📚 단축 URL 생성 API 연동 가이드

POST /api/v1/links

지정된 목적지 URL을 스마트 라우팅 및 수익형 옵션과 함께 초고속 단축 URL로 가공하여 반환합니다.
인증을 위해 HTTP Request Header에 X-API-KEY 명칭으로 발급받은 API 키를 실어 호출해야 합니다.

📋 Request Headers
헤더명 필수 여부 설명
Content-Type 필수 application/json
X-API-KEY 필수 발급받은 활성 API Key (예: pxl_live_abc123...)
💻 Python 호출 예제
import requests

url = "http://localhost:8090/api/v1/links"
headers = {
    "X-API-KEY": "pxl_live_YOUR_API_KEY_HERE",
    "Content-Type": "application/json"
}
payload = {
    "defaultTargetUrl": "https://example.com/products/123",
    "title": "Summer Sale Campaign",
    "description": "Get 30% discount immediately!",
    "isAdEnabled": True,
    "adTimerSeconds": 5
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
