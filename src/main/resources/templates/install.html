Pixel-Link

최초 시스템 설치 및 최고 관리자(ADMIN) 설정

에러 메시지 내용
[설치 진행 가이드]
1. 설치가 완료되면 시스템 상태 테이블(`system_settings`)의 초기 플래그가 `is_installed = 'true'`로 업데이트됩니다.
2. 설치 완료 이후에는 보안을 위해 본 설치 화면(`/app/install`)에는 **다시는 접속할 수 없도록 원천 차단**됩니다.
3. 설치와 관련된 메타데이터(IP 주소, 브라우저 환경, 시간)는 보안 감사 로그 테이블(`system_audit_logs`)에 기록으로 보관됩니다.
