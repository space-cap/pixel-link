# Pixel-Link UI/UX 디자인 가이드라인

본 문서는 Pixel-Link 서비스의 화면 디자인 일관성과 프리미엄 감성(Premium Aesthetic)을 유지하기 위해 정의된 UI/UX 스펙 문서입니다. HTML5/Vanilla CSS(CSS 모듈) 기반의 디자인 시스템 토큰과 레이아웃 상세를 포함합니다.

---

## 1. 디자인 시스템 토큰 (CSS Variables)

신뢰와 전문성을 주는 깨끗하고 모던한 라이트 테마(Light Mode)를 기조로 하며, 부드러운 화이트 레이어와 맑은 로열 블루 및 차분한 네이비 톤을 조합하여 사용자에게 높은 보안성과 신뢰감을 전달합니다.

```css
/* src/main/resources/static/css/global.css 에 내장될 토큰 */
:root {
  /* 색상 팔레트 (HSL/HEX) */
  --bg-main: #f8fafc;            /* 연한 그레이시 슬레이트 백그라운드 */
  --bg-card: #ffffff;            /* 깔끔한 카드용 순백색 */
  --accent-primary: #1d4ed8;     /* 신뢰감을 주는 깊은 로열 블루 */
  --accent-secondary: #4f46e5;   /* 보조 네이비/인디고 퍼플 */
  
  /* 전경색 (텍스트) */
  --text-main: #0f172a;          /* 가독성 높은 딥 슬레이트 (#0F172A) */
  --text-muted: #475569;         /* 보조 및 본문 텍스트 슬레이트 */
  --text-dark: #94a3b8;          /* 플레이스홀더 및 비활성 회색 */

  /* 피드백 색상 */
  --color-success: #059669;      /* 그린 (성공/복사 완료) */
  --color-error: #dc2626;        /* 레드 (실패/차단) */

  /* 카드 섀도우 및 보더 */
  --card-border: rgba(15, 23, 42, 0.08);
  --card-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.05), 0 10px 15px -3px rgba(0, 0, 0, 0.08);

  /* 테두리 둥글기 (Border Radius) */
  --radius-sm: 8px;
  --radius-md: 12px;
  --radius-lg: 16px;
  
  /* 애니메이션 속도 */
  --transition-fast: 0.15s cubic-bezier(0.4, 0, 0.2, 1);
  --transition-normal: 0.25s cubic-bezier(0.4, 0, 0.2, 1);
}
```

## 2. 타이포그래피 (Typography)

* **웹 폰트**: Google Fonts에서 **Inter**(본문용) 및 **Outfit**(제목/대시보드 헤더용)을 로드하여 사용합니다.
* **계층 구조 (Font Scale)**:
  * 대시보드 로고: `28px (ExtraBold)`
  * 카드/섹션 타이틀: `18px (SemiBold)`
  * 입력 필드 라벨 / 본문: `14px (Regular / Medium)`
  * 보조 텍스트 (클릭 로그 등): `12px (Regular)`

## 3. 화면별 레이아웃 명세

### 3.1. [1단계] 초고속 리다이렉트 대기 화면 (`redirect.html`)
사용자가 단축 URL 클릭 시 최초로 보게 되는 스크린입니다. 이탈률을 줄이기 위해 로딩이 부드럽고 시각적으로 매우 수려해야 합니다.

* **레이아웃**: 화면 정중앙 배치 (Flexbox / `justify-content: center; align-items: center;`)
* **디자인 요소**:
  1. **배경**: 미세한 방사형 그라데이션 적용 (`radial-gradient(circle at center, #0F172A 0%, #020617 100%)`)
  2. **중앙 로더**: 크기 `48px * 48px`의 회전형 스피너. 그라데이션 처리 (`linear-gradient(var(--accent-primary), var(--accent-secondary))`)
  3. **안내 문구**: "안전한 목적지로 연결 중입니다..." 텍스트는 `var(--text-muted)` 색상으로 차분하게 노출.
* **UX 모션**: 리다이렉트 실행 시 화면 전체가 페이드아웃(`opacity: 0`) 되며 부드럽게 이행하는 시각 효과 부여.

---

### 3.2. [1단계] 관리자 대시보드 (`dashboard.html`)
마케터가 단축 링크를 등록하고 클릭 수를 모니터링하는 컨트롤 타워입니다.

#### 레이아웃 구조
* **Header**: 상단 고정식 글래스모피즘 바 (서비스 로고 `Pixel-Link`, 사용자 프로필, 로그아웃 버튼).
* **Main Area**: `max-width: 1200px` 중앙 정렬 그리드 레이아웃.
  * **좌측 영역**: 단축 링크 생성 폼 (2단계 스마트 라우팅 입력 필드 확장 고려).
  * **우측 영역**: 실시간 링크 리스트 및 간이 통계 카드.

#### 상세 컴포넌트 명세

```
+-------------------------------------------------------------+
|  [PIXEL-LINK] Logo                               Profile    |  <- Header
+-------------------------------------------------------------+
|                                                             |
|  +--------------------+  +-------------------------------+  |
|  | 단축 링크 생성 폼  |  | 통계 요약 (오늘 클릭 / 전체)  |  |  <- Main
|  |                    |  +-------------------------------+  |
|  | - 목적지 URL       |  | 단축 링크 카드 목록           |  |
|  | - 맞춤형 주소      |  |                               |  |
|  | - SEO 설정         |  | [ s/my-promo ]   420 Clicks   |  |
|  | - 픽셀 설정        |  | Target: https://example.com   |  |
|  |                    |  | [복사] [수정] [삭제]           |  |
|  | [ 링크 생성하기 ]  |  +-------------------------------+  |
|  +--------------------+                                     |
+-------------------------------------------------------------+
```

1. **단축 링크 생성 폼 (글래스모피즘 카드)**
   * `background: var(--glass-bg); backdrop-filter: var(--glass-blur); border: 1px solid var(--glass-border);`
   * 입력 필드는 배경색을 반투명 검은색으로 처리하고, 포커스 시 테두리가 `var(--accent-primary)` 색상으로 밝아지며 글로우 효과(shadow glow)가 퍼집니다.
2. **단축 링크 카드**
   * 각 카드는 단축 주소명, 타겟 주소, 생성일, 그리고 **활성화된 픽셀 아이콘 표시**(예: Facebook `F`, Google `G` 뱃지 노출)를 포함합니다.
   * 카드 위에 마우스를 올리면(Hover), 카드가 미세하게 위로 올라가고(`transform: translateY(-4px)`) 보더 색상이 밝아지는 호버 마이크로 모션 제공.

## 4. 핵심 UX 인터랙션 및 애니메이션 피드백

### 4.1. 단축 링크 원클릭 복사 기능
* **동작**: 유저가 대시보드에서 [복사] 버튼을 누르면 단축 URL이 클립보드에 저장됩니다.
* **피드백**: 버튼의 텍스트가 "복사"에서 "복사 완료! ✔"로 변경되며 배경색이 성공 초록색(`var(--color-success)`)으로 채워진 후 1.5초 뒤 부드럽게 원래 상태로 롤백됩니다.

### 4.2. 버튼 호버 및 액션 피드백
* 모든 버튼은 마우스 호버 시 스케일이 약간 커지며(`transform: scale(1.02)`), 클릭 시에는 살짝 눌리는 듯한(`transform: scale(0.98)`) 느낌을 주어 아날로그 장치를 만지는 듯한 쫀득한 터치감을 구현합니다.
